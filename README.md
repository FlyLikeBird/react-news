# React技术栈实现新闻平台

> 项目地址：https://github.com/FlyLikeBird/react-news

> 部署在阿里云地址：http://www.renshanhang.site

## 安装步骤
    
    # 克隆项目仓库到本地
    git clone ...
    
    # 安装项目依赖
    npm i (or yarn)
    
    # 开启本地服务http://localhost:8081即可运行
    npm start
